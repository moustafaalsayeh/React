# React App
Frontend Web Development with React.

# Get Started
```
git clone https://github.com/moustafaalsayeh/React.git

Install these libraries if not avaliable.
npm install react-scripts --save
npm install -g json-server

To start the Json Server open your teminal go to the the react project directory, you will find json-server directory open it and command
>>> json-server --watch db.json -p 3001 -d 2000

To start the project open your teminal go to the the react project directory
>>> cd React
>>> npm start
```
